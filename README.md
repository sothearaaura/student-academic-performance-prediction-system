# Student Academic Performance Prediction System

## Overview

A full-stack web application that predicts student academic performance using Machine Learning.

## Features

- User Authentication
- Student Management
- Academic Performance Prediction
- Dashboard & Analytics
- Reports
- Prediction History

## Technology Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy

### Database
- MySQL

### Machine Learning
- Scikit-learn
- Joblib
